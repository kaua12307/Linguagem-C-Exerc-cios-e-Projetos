# Linguagem-C-Exerc-cios-e-Projetos
Repositório dedicado aos meus estudos em Linguagem C, contendo exercícios, desafios e projetos desenvolvidos durante o aprendizado. Aqui serão armazenadas implementações voltadas para lógica de programação, estruturas de dados, algoritmos, manipulação de memória e desenvolvimento de aplicações no terminal, acompanhando minha evolução na linguagem.
